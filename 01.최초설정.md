# Git 최초 설정
git  config --global user.name "사용자이름"
git  config --global user.email "사용자이메일"

## git option
git config -- system : 시스템 설정 => 모든 사용자 및 저장소에 적용
git config -- global : 전역 설정   => 모든 저장소에 적용
git config -- locak  : 로컬 설정   => 현재 저장소에만 적용

git config --global user.name MoaisMoa
git config --global user.email mom.moom.mu@gmail.com

# Git Clone
원격 저장소를 로컬 저장소로 복제하는 명령어 입니다.

git clone <원격저장소 URL>
    : https://github.com/MoaisMoa/MSA15_GIT.git

# git add
    : 파일을 스테이징 영역에 추가하는 명령어입니다.

스테이징
    : 변경사항을 적용할 영역

변경 사항을 스테이징 영역에 반영한다.

## 파일 스테이징
git add <파일명>

## 경로 스테이징
git add <경로명>

## 전페 파일 스테이징
git add .

# Push
로컬 저장소의 커밋을 원격 저장소에 업로드하는 작업

### 저장소 및 브랜치 지정
git push <원격 저장소 이름> <브랜치 이름>

### 연결된 원격 저장소 및 브랜치로 푸시
git push

# Pull
원격 저장소의 커밋을 로컬 저장소로 가져오는 명령어

### 저장소 및 브랜치 지정
git pull <원격 저장소 이름> <브랜치 이름>

### 연결된 원격 저장소 및 브랜치에서 가져오기
git puu
